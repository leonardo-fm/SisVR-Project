# SisVR – Gaussian Splatting VR Project

This project was developed for the University of Turin (UNITO) as part of the **Sistemi di Realtà Virtuale (SisVR)** course.  
The project focuses on exploring Gaussian Splatting visualization inside a Virtual Reality environment using Unity.

## Setup and Usage

To run the project, download or clone this repository and open it with Unity.  
Once the project is loaded, open the scene named **GS** and press Play to start the application.

The project has been tested using **Meta Quest 2**, but it is expected to work with other VR headsets compatible with Unity and OpenXR as well.

## Gaussian Splatting

The VR Gaussian Splatting implementation used in this project is based on the following open-source repository:
